<h1><span style="color: #000080;">Data types:</span></h1>
<p>Data type is an important topic in programming.&nbsp;</p>
<p>We described the variables as containers which stores data. The stored data can be from different types and, each type has some characteristics that make the programmer use them for different purposes.</p>

<p>Python provides some built-in data types by default:</p>
<ul>

<ul>
<li><span style="color: #800000;">str</span></li>
</ul>
</li>

<ul>

</li>
<li><strong>Set Types:</strong>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;&nbsp;
