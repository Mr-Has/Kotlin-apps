<h1><span style="color: #000080;">Escape character:</span></h1>
<p>There are some characters which are reserved by Python (<span style="color: #800000;">=</span>, <span style="color: #800000;">&amp;</span>, etc.) and some others that the programmer cannot use in a string because it will interrupt the program syntax (<span style="color: #800000;">"</span>, <span style="color: #800000;">'</span>, <span style="color: #800000;">/</span>, etc.).</p>
<p>Python overcome this issue by reserving the backslash / character as an escape character.</p>
<p>Basically, the character after backslash will lose its functionality and become a printable character in a string.&nbsp;</p>
<p><strong>For example:</strong></p>
<table style="margin: auto; box-shadow: 3px 3px 10px #000;" border="1">
<tbody>
<tr style="background-color: rgba(0,0,0,0.1);">
<td>
<p>text = <span style="color: #008000;">"Today was a "</span>cold<span style="color: #008000;">" day"</span></p>
<p>text = <span style="color: #008000;">"Today was a</span> \<span style="color: #008000;">"cold</span>\<span style="color: #008000;">" day"</span></p>
</td>
</tr>
</tbody>
</table>
