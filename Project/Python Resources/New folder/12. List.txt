<h1><span style="color: #000080;">List in Python:</span></h1>
<p>List is an ordered collection of elements (values).</p>
<p><strong>Note:</strong></p>
<p>The name element is the same as item and they refer to each value in a list.</p>
<p><strong>Recall:</strong></p>
<p>String is a collection of ordered characters.&nbsp;</p>
<p>List is the same as string, and it is a collection of ordered characters, but unlike string, the characters of a list are values, and they can be from any data type.</p>
<p>We define a list by square brackets <span style="color: #800000;">[ ]</span> and the items are separated by comma&nbsp;( <span style="color: #800000;">,</span> ).</p>
<p><strong>For example:</strong></p>

<h2><span style="color: #000080;"><strong>Ordered:</strong></span></h2>
<p>The items in the list are ordered, and we can access them by their index.</p>
<p>List allows duplicated items because list is a collection of indexes.</p>
<p><strong>For example:</strong></p>
<table style="margin: auto; box-shadow: 3px 3px 10px #000;" border="1">
<tbody>
<tr style="background-color: rgba(0,0,0,0.1);">

</tr>
</tbody>
</table>
<p><strong>Note:</strong></p>
<p>The index of first item is 0.</p>
