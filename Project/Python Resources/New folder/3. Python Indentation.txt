<h1><span style="color: #000080;">Python Indentation:</span></h1>
<p>Indentation is the spaces left empty at the beginning of a code line.</p>
<p>Python indentation purpose is to indicate the <strong>block of code</strong>, therefore indentation is very important in Python.&nbsp;</p>
<p><strong>For example:</strong></p>

</td>

<h2><span style="color: #000080;">Comments:</span></h2>
<p>The best practice to keep the line of codes organized is by describing the block of codes in a comment which describe the code.</p>
<p>For this purpose, Python has a comment capability.</p>
<p>Any line starts with # will be considered as a comment and Python will not execute it.&nbsp;</p>
<p><strong>For example:</strong></p>
<table style="margin: auto; box-shadow: 3px 3px 10px #000;" border="1">
<tbody>
<tr style="background-color: rgba(0,0,0,0.1);">
<td>
<p><span style="color: #808080;"># This is a comment, and it won't be executed.</span></p>
<p><span style="color: #808080;"># This is the second comment.&nbsp;</span></p>
</td>
</tr>
</tbody>
</table>
