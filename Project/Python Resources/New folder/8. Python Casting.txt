<h1><span style="color: #000080;">Casting in Python:</span></h1>
<p>Python allows the programmer to specify the data type by adding constructor. These types are pre-defined in Python.</p>
<p><strong>These constructors are:</strong></p>
<h2><span style="color: #000080;">int():</span></h2>
<p><span style="color: #800000;">int()</span> construct a whole number.&nbsp;</p>
<p>If we pass number with decimal to this constructor, it will round it down to the previous integer number.</p>
<p>It will output the number value if we pass string to this constructor.</p>
<p><strong><span style="color: #000000;">Note:</span></strong></p>
<p>The string passed to these constructors must represent Numeric values.</p>
<p><strong>For example:</strong></p>
