<h1><span style="color: #000080;">Boolean variables:</span></h1>
<p>Boolean data type is either True or False. Python reserves True and False keywords for Boolean variables.</p>
<p>By using Boolean we can evaluate an expression to find if it is True of False.</p>
<p><strong>For example:</strong></p>

<tbody>
<tr style="background-color: rgba(0,0,0,0.1);">
<td>

<p><span style="color: #800000;">print</span>(<span style="color: #008000;">110</span> == <span style="color: #008000;">110</span>)</p>
<p><span style="color: #800000;">print</span>(<span style="color: #008000;">5</span> &gt; <span style="color: #008000;">12</span>)</p>

</tr>
</tbody>
</table>
