<p>Python divides operators in the following categories:</p>
<ul>
<li><strong>Arithmetic operators</strong></li>
<li><strong>Assignment operators</strong></li>
<li><strong>Comparison operators</strong></li>
<li><strong>Logical operators</strong></li>
<li><strong>Identity operators</strong></li>
<li><strong>Membership operators</strong></li>
<li><strong>Bitwise operators</strong></li>
</ul>

<h2><span style="color: #000080;">Bitwise operators:</span></h2>
<p>Bitwise operators are responsible to compare binary numbers.</p>
  <html>
<head>
<style>
table, th, td {
  border: 1px solid black;
}
</style>
</head>
<body>
  </body>
</html>
<table style="width: 100%;">
<tbody>
<tr>
<th>Operator</th>
<th><strong>Description</strong></th>
<th>Name</th>
</tr>
<tr>
<td><span style="color: #800000;">&amp;&nbsp;</span></td>
<td>Sets each bit to 1 if both bits are 1.</td>
<td>bitwise <span style="color: #800000;">AND</span></td>
</tr>
<tr>
<td><span style="color: #800000;">|</span></td>
<td>Sets each bit to 1 if one of two bits is 1.</td>
<td>bitwise <span style="color: #800000;">OR</span></td>
</tr>
<tr>
<td><span style="color: #800000;">^</span></td>
<td>Sets each bit to 1 if only one of two bits is 1.</td>
<td>bitwise <span style="color: #800000;">XOR</span></td>
</tr>
<tr>
<td><span style="color: #800000;">~&nbsp;</span></td>
<td>Inverts all the bits.</td>
<td>bitwise <span style="color: #800000;">NOT</span></td>
</tr>
<tr>
<td><span style="color: #800000;">&lt;&lt;</span></td>
<td>Shift left by pushing zeros in from the right and let the leftmost bits fall off.</td>
<td>bitwise left shift</td>
</tr>
<tr>
<td><span style="color: #800000;">&gt;&gt;</span></td>
<td>Shift right by pushing copies of the leftmost bit in from the left, and let the rightmost bits fall off.</td>
<td>bitwise right shift</td>
</tr>
</tbody>
</table>
